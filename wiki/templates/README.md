# Wiki Templates

목적: LLM이 새 페이지를 만들 때 따라야 할 기본 뼈대를 모아 둔다.

## Summary
- `wiki/templates/`는 페이지 작성 규칙을 코드처럼 재사용하기 위한 자리다.
- 새 source, topic, entity, decision 페이지는 가능한 한 이 템플릿을 따른다.

## Key Points
- 모든 페이지는 짧은 목적 문장으로 시작한다.
- 사실, 해석, 미해결 질문을 분리해서 적는다.
- `wiki/index.md`와 `wiki/log.md`는 항상 함께 갱신한다.

## Details
- `source.md`: 단일 소스의 요약과 핵심 사실.
- `topic.md`: 여러 소스를 묶어 만드는 개념/주제 페이지.
- `entity.md`: 사람, 조직, 제품 같은 실체 페이지.
- `decision.md`: 운영/아키텍처/정책 결정 기록.
